HOW TO LEARN CODING

By Kingsley Cheng

1. Easiest programming language: Python / Javascript
2. Learn by Coding (NOT watching people): Use Codecademy, Freecodecamp.org, w3schools etc.
3. Building own projects give benefits: Debugging, give confidence
4. Project Examples: Web Crawler, Alarm Clock, Wikipedia Article of the Day, Spam Filter etc.
5. How to Prepare for Questions: Data Structures and Algorithms, free course on Udacity
6. Must finish online courses and make progress EVERY DAY
7. Implement the algorithms and finish the assignments
8. Use LeetCode: Interview-based Questions
9. DO NOT QUIT EASILY

That's it :)
